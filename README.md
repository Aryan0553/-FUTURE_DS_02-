1. Short Description
This project analyzes social media marketing campaign data to evaluate performance, engagement, sales, and ROI. 
Using Python and Power BI, the project transforms raw campaign data into actionable insights through KPIs, visual analysis, and interactive dashboards to support data-driven marketing decisions.
